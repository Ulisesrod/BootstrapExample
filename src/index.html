Programación Web I

Reseña del Curso

Este curso de programación procura que la persona estudiante desarrolle su habilidad para el desarrollo de aplicaciones de software utilizando ambientes web acorde con los estándares internacionales, los patrones de diseño y las buenas prácticas de la industria, que permitan la visualización, manejo y acceso seguro al sitio web desde cualquier tipo de dispositivo.
